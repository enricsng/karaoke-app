# Mini Karaoke App

A lightweight karaoke app built with React and Node

## Features
- Direct playback from YouTube
- Playback controls from secondary device

## How to run

1. Clone repo
2. Install dependencies `npm install`
3. Start server `npm start `
4. Go to http://localhost:3000/frontend.html for player, http://localhost:3000/controller.html for controller
